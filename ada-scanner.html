ADA Scanner

WCAG2AA / Section 508 accessibility scanning, wrapped for CI.

AccessibilityPythonCLI

ADA Scanner is a Python wrapper around the Pa11y CLI, built to make automated WCAG2AA and Section 508 accessibility scanning easy to run and easy to wire into a build pipeline.

What It Does

The scanner (runner.py) accepts either a full site crawl or a JSON list of specific URLs to check, then runs each page through Pa11y and reports the results. It's packaged with PyInstaller for standalone distribution and integrates with Jenkins CI, so accessibility regressions can be caught automatically as part of a normal build, not just during manual audits.
