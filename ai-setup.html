Local AI Setup

A portable, offline-friendly Claude Code + local LLM dev environment.

Dev ToolingShell / Config

Local AI Setup is a collection of scripts and config files for running Claude Code alongside a local Ollama/LiteLLM model stack — useful for working offline, on-prem, or with self-hosted models instead of (or alongside) hosted APIs.

What's Included

  • zshrc-snippets.sh — shell setup for the environment.
  • bin/oclaude — launcher that points Claude Code at a local model backend.
  • litellm_config.yaml / custom_models.yaml — LiteLLM routing and model definitions.
  • run-ai / patch-ai — an orchestrator loop: generate, test, fix, and escalate automatically.
  • claude-bootstrap — symlinking helper for consistent config across machines.
  • wt — a git worktree helper for parallel branches.
