Tapirus Studio

Independent software, built one focused tool at a time.

Tapirus Studio is a solo software practice making small, purposeful tools — an app, a training tool, a testing utility, and a dev environment setup — each built to solve one real problem well.

  • PulseStrum

    A visual and audio rhythm trainer for guitar. Stop counting the beat — see and hear it.

  • fut.io — FUT Randomizer

    A cognitive-load training tool for soccer players: randomized numbers, shapes, and timers to train scanning under pressure.

  • ADA Scanner

    A Python wrapper around Pa11y for WCAG2AA / Section 508 accessibility scanning, with CI integration.

  • Local AI Setup

    A portable Claude Code + local Ollama/LiteLLM development environment for working offline or on-prem.
